# burpsuite-http-history-xml-convert-html

---

## Overview

---

**burpsuite-http-history-xml-convert-html**는 burpsuite의 HTTP HISTORY를 xml 파일을 HTML에서 확인하도록 변환해주는 도구입니다.

![Test Result](./img/test_result.png)

## Usage

---

```bash
# help
python main.py -h

# xml to html
python main.py ./test/test.xml
```
